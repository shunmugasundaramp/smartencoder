AKKU::Smart::Encoder Documentation

AKKU::Smart::Encoder, Release 2.0.0

This is AKKU::Smart::Encoder, a C++ library for do the encoding(either XML or JSON) written by Shunmuga Sundaram. It is a independent source which should not having any other thirdparty software references

Currently it is having a support to compile with GCC Compiler. Hopefully it will come for other compilers too.

  4. Enhanced to any new datatype

Usage

In Linux

To Compile

./linbuild.sh

To Run

./objs/usage.exe

In Windows

To Compile

winbuild.bat

To Run

objs\usage.exe
